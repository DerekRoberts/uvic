
Please run 'make' (no quotes) from a Linux command prompt to compile this program.

Syntax:

    ./diskinfo <DISK.IMA>
    ./disklist <DISK.IMA>
    ./diskget <DISK.IMA> <FILE.TXT>
    ./diskput <DISK.IMA> <FILE.TXT>

<DISK.IMA> may be any FAT12 formatted disk image
<FILE.TXT> may be any file requested to be passed to or from that image
*note: any letters in filename <FILE.TXT> must be in CAPITAL LETTERS

Please run 'make clean' (no quotes) to remove the created files.
